# Gerador de Códigos - Bot Discord

## Como Usar

1. **Edite bot.js** - Coloque seu token do Discord
2. **Execute:** `npm install`
3. **Execute:** `node bot.js`
4. **No Discord:** `/getkey` para obter código
5. **No site:** Gere códigos em 13/01/2026 12:59:01

## Arquivos Criados

- index.html - Site principal
- css/style.css - Estilos
- js/script.js - Lógica do site
- bot.js - Bot Discord
- package.json - Dependências

## Deploy no GitHub Pages

1. Faça push para o GitHub
2. Vá em Settings > Pages
3. Selecione branch main
4. Selecione pasta / (root)
5. Salve

Pronto!
